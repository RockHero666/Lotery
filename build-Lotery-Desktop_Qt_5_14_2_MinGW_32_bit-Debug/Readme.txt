p1,2,3,4,5   отвечает за паттерн поворота на N градусов картинки шара при ударе об стенку 
cfg_x_inc и cfg_y_inc   отвечают за скорость анимации (прокрутки шара) 
physi отыечает за скорость перемещения шара по экрану из расчета  X пикселей за фрейм (лотерея работает в 100FPS)  default = 100*4 = шар преодолеет 400 пикселей в секунду
predel верхний предел возможного выигрыша  ( по стандарту 2    то есть больше чем 2 совпадения невозможно) ( принимает значения от 0-6   ДРУГИЕ НЕ СТАВИТЬ)
Все параметры принимают  положительные и отрицательные числа, все параметры принимают рацианальные числа( 1, 0.25 , -23)
Любое значение больше 1000 лучше не ставить или прога упадет или комп
Если все поламалось и нечего не работает  - удалить settings.ini
